Ratio Selection ▾

5:1 won at 4.83/5 -- best compromise between transparency (low ratio = more backdrivable) and torque multiplication.

Bearing Selection ▾

Ball bearings at 4.05/5. Cost weighted at 60% reflecting prototype budget. Meets radial load requirements while staying under $120 total.

Key Insight

The #1 failure mode in student robotics is not the gearbox itself, but how it attaches to the motor. Interface definition was prioritized accordingly.

02 — CAD Architecture

CATIA V6 Parametric Design

~60-80 hours learning CATIA V6 from scratch. The entire assembly is skeleton-driven: a master skeleton holds all parameters, and parts read from it through formulas -- never referencing each other directly.

Core Principles ▾
  • Single source of truth -- all positioning in skeleton
  • No B-Rep references -- edges/faces break on feature reorder
  • Body-level patterning -- more stable than feature-level
  • Placeholder gears -- define interface geometry, swap in real teeth later
  • Chain-of-zeros -- zero offsets at every level eliminates positioning drift

03 — DFM & Tolerancing

FDM Calibration Pipeline

ISO fit designations describe intent. FDM offsets approximate those fits. Calibration coupons validate every interface before committing to a full build.

Design Decision

All bearing interfaces target finger-press transition fits. FDM surface texture provides grip, so press fits are unnecessary for a backdrivable actuator.
