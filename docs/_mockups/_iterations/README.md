# QDD Project Page — Mockup Iterations

Exploratory mockups for the QDD Gearbox project page on aaronevans.ca. These informed the final design direction.

| File | Concept | Feedback |
|------|---------|----------|
| mockup-1-cards | Dark card grid + timeline bar | Liked — good attention grabber |
| mockup-2-scroll | Full-width alternating light/dark sections | Liked — good content depth |
| mockup-3-sidebar | Sidebar nav + accordions | Liked — clear navigation, clean list |
| mockup-4-final | Cards + sidebar hybrid | Rejected — too blog-like, too much scrolling |
| mockup-5-onepager | One-pager with hero | Rejected — hero eats viewport, too much blank space |
| mockup-6a-bento | Bento grid, asymmetric tiles | Explored |
| mockup-6b-split | Split screen, hero left, info right | Explored |
| mockup-6c-magazine | Magazine spread, floating cards | Liked as concept — led to mockup 7 |
| mockup-7-interactive | Click-to-explore, central display swap | Liked concept but side icons too small/text-only |

**Design direction chosen:** Hybrid of mockup-3 sidebar nav + mockup-2 content sections, using the homepage-v7 design system (fonts, colors, spec chips, patterns).
