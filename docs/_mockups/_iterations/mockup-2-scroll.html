02 — CAD Architecture

Skeleton-Driven Design in CATIA V6

~60-80 hours learning CATIA V6 from scratch. The assembly is driven by a master skeleton containing all parameters — parts never reference each other directly. Change one skeleton value, everything updates.

Key principles: single source of truth, no B-Rep references, body-level patterning, placeholder gears as geometric contracts. Chain-of-zeros positioning eliminates offset errors across sub-assemblies.

03 — DFM & Tolerancing

FDM Calibration Pipeline

ISO fit designations describe intent. FDM offsets approximate those fits. Every bearing interface has a named offset parameter in the skeleton, validated by calibration coupons before the full build.

All bearing interfaces target finger-press transition fits. FDM surface texture provides grip, so press fits are unnecessary for a backdrivable actuator. Self-tapping threads validated at M3-M5 as simpler alternative to heat-set inserts.
