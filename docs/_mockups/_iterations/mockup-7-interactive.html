CAD Architecture
CATIA V6 Parametric Design
~60-80 hours learning from scratch. Skeleton-driven -- every part reads from a single source of truth.
[ CATIA: full assembly cross-section with gear mesh visible ]
  • Single source of truth -- all positioning in master skeleton
  • No B-Rep references -- sketch on planes, reference wireframe only
  • Body-level patterning -- more stable than feature-level
  • Chain-of-zeros -- zero offsets at every hierarchy level
  • Placeholder gears -- geometric contracts, real teeth swap in
