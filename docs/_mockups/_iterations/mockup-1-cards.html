CAD Architecture

CATIA V6 Parametric Design

Skeleton-driven assembly where every part reads from a single source of truth. ~60-80 hours learning from scratch.

Core principles:

  • All positioning in master skeleton — parts never reference each other
  • No B-Rep references (edges/faces break on reorder)
  • Body-level patterning for stability
  • Placeholder gears as geometric contracts — real teeth swap in

Chain-of-zeros positioning: skeleton at origin, sub-assemblies constrained to skeleton midplane, parts to sub-assembly origin. Zero offsets everywhere.

DFM

FDM Tolerancing & Calibration

ISO fit intent translated to FDM offsets. Calibration coupons validate every bearing interface before the full build.

All bearing interfaces target finger-press transition fit — FDM surface texture provides grip, so press fits aren't needed for a backdrivable actuator.

InterfaceOffsetStatus
Main bearing bore+0.10mmValidated
Carrier bearing bore+0.10mmApplied

Self-tapping validated: M3=2.6mm, M4=3.6mm, M5=4.6mm pilots. Simpler than heat inserts for prototyping.

Iteration

Rev 00A → Rev 00B

First prototype built, tested, and 13 design changes scoped for the next revision based on test data.

Rev 00B must-haves (from test data):

  • Spur gears (eliminates herringbone assembly error)
  • Larger output shaft + bearing
  • Fix planet bearing shoulder print quality
  • Decouple lid constraint from carrier
  • Self-tap threads (drop heat inserts)
