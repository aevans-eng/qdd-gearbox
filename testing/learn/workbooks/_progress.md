# QDD Learning System — Progress Tracker

> **Goal:** Master all QDD project knowledge for a Tesla Optimus actuator design interview.
> Work sequentially. Bring completed workbooks to Claude: **"Grade my workbook XX"**

## How it works

1. Open the workbook, read questions open-book, write your answers in the response areas
2. Check one confidence box per question (High / Medium / Low)
3. Bring the completed file to Claude for written grading + live follow-up review
4. Update this tracker with your results

**Status flow:** `Not started` → `In progress` → `Grading` → `Live review` → `Complete`
**Grade format:** Solid / Partial / Redo count (e.g., "8S 2P 0R")
