# 2026-01-15 - Sun Gear Sweep (S=4 to S=7) Targeting 5:1 Ratio

## Iteration ID
`2026-01-15_sun_sweep_5to1`

## Objective
Sweep sun gear tooth count from 4 to 7 while maintaining approximately 5:1 gear ratio. Generate labeled PNGs showing each configuration.

## Kinematic Configuration

Changed from the initial setup to achieve a reduction ratio:

| Config | Input | Output | Fixed | Ratio Formula |
|--------|-------|--------|-------|---------------|
| Initial (`'c', 'r', 's'`) | carrier | ring | sun | R/(R+S) < 1 (speed increase) |
| **This sweep** (`'s', 'c', 'r'`) | sun | carrier | ring | (R+S)/S = 1 + R/S (reduction) |

For 5:1 reduction: `1 + R/S = 5` → `R = 4S`

## Gear Meshing Constraint

For a planetary gear to mesh properly: `R = S + 2P`

Combining with R = 4S for 5:1 ratio:
```
4S = S + 2P
3S = 2P
P = 1.5S
```

This means P is only an integer when S is even. For odd S values, we must round P and accept a slight ratio deviation.

## Configurations Tested

### S=4 (Sun = 4 teeth)
- **Calculated:** R = 4×4 = 16, P = 1.5×4 = 6
- **Used:** R=16, P=6, S=4
- **Number of planets (N):** 4
- **Tooth profile (b):** 0.8
- **Actual ratio:** (16+4)/4 = 20/4 = **5.000:1** (exact)
- **Meshing check:** R = S + 2P → 16 = 4 + 12 = 16 ✓
- **Output file:** `pngs/2026-01-15_sun_sweep_5to1_S4_P6_R16_N4_ratio5.00.png`

- **Decision:** Rounded P up to 8 for ratio closer to 5.0 from above
- **Used:** R=21, P=8, S=5

- **Tooth profile (b):** 0.8

## Number of Planets (N) Selection

The original configuration used N=5 planets. For this sweep:
- S=4: Used N=4 (R+S=20 is divisible by 4)
- S=5,6,7: Used N=3 (works well with larger planet gears)

The number of planets affects:
1. Load distribution (more planets = more load sharing)
2. Physical clearance (must fit without overlapping)
3. Torque capacity

## Observations

1. **Even sun teeth (S=4, S=6) give exact 5:1 ratio** - Because P = 1.5S is then an integer
2. **Odd sun teeth require ratio compromise** - Must round planet teeth up or down
3. **Larger sun gears have more teeth in mesh** - May provide smoother operation
4. **Smaller sun gears are more compact** - S=4 gives smallest overall gear train for same ratio
5. **Planet count varies** - Larger planets (from larger S) require fewer planets to avoid overlap

## Visual Comparison Notes

- S=6: 3 planets, more defined tooth profiles on all gears
- S=7: 3 planets, largest sun gear, most teeth visible, potentially smoothest mesh
