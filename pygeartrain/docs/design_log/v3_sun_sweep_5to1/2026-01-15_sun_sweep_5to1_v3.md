## Critical Discovery: The b Parameter

**ROOT CAUSE OF v1/v2 FAILURES:**
The `b` parameter controls the ratio of epicycloid to hypocycloid in the tooth profile.
- `b=0.8` (used in v1/v2): Creates tall teeth that overlap
- `b=0.5` (v3): Creates shorter teeth that mesh properly

This was discovered through Shapely polygon intersection analysis showing:
- S=4, P=6, R=16, N=3 with b=0.8: **5.68% sun-planet overlap** (INVALID)
- S=4, P=6, R=16, N=4 with b=0.5: **0.00% overlap** (VALID)

## Validation Methodology

Created `gear_validator.py` and `find_valid_gears.py` modules that:

1. **Convert gear profiles to Shapely polygons**
2. **Check polygon intersection area** between:
   - Sun gear and each planet gear
   - Planet gears with each other
3. **Test multiple rotation phases** (0, 0.25, 0.5) to catch worst-case positions
4. **Accept configurations with <1% overlap** (allows numerical tolerance)

### Validation Code Summary
```python
def check_overlap(S, P, R, N, b):
    # Get Shapely polygons from gear profiles
    sun_poly, planet_polys, ring_poly = get_polygons(S, P, R, N, b, phase)

    # Check sun-planet intersection
    intersection = sun_poly.intersection(planet_poly)
    overlap_pct = (intersection.area / sun_poly.area) * 100

    # Valid if overlap < 1%
    return overlap_pct < 1.0
```

## Design Rules Discovered

### Rule 1: b Parameter Controls Tooth Height
| b Value | Tooth Style | Result |
|---------|-------------|--------|
| 0.8 | Tall epicycloidal | Often overlaps |
| 0.7 | Medium-tall | Sometimes overlaps |
| 0.5 | Balanced | Usually valid |
| 0.3 | Short hypocycloidal | Always valid (but weaker) |

**Recommendation:** Use `b=0.5` as default for safe meshing.

### Rule 2: Sun-Planet Overlap is the Critical Check
Previous checks focused on planet-to-planet spacing. The actual failure mode is:
- **Planet teeth cutting through sun teeth**
- Visible as filled polygon intersection area > 0

### Rule 3: More Teeth = More Flexibility
Larger sun gears (more teeth) allow more planets and higher b values:
- S=4: Only N=4 or N=2 work
- S=5: Only N=2 works
- S=6: N=2 or N=3 work (sweet spot)
- S=7+: Generally N=2 works

### Rule 4: Even Sun Teeth Give Exact 5:1 Ratio
For ratio = (R+S)/S = 5, need R = 4S.
With R = S + 2P constraint: P = 1.5S (integer only when S is even).

## Locked Gear Configuration

**RING LOCKED, SUN INPUT, CARRIER OUTPUT**

| Element | Role | Connection |
|---------|------|------------|
| Ring | LOCKED | Bolted to housing/frame |
| Sun | INPUT | Connected to motor shaft |
| Carrier | OUTPUT | Connected to load |

**Ratio Formula:** `(R + S) / S = 1 + R/S`

Other configurations produce different ratios (see v2 documentation).
